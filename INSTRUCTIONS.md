# Data Science Case Study: Loan Repayment Analysis

## Repo Structure

```

├── data
│   ├── external    <- Data from third party sources. (added in .gitignore)
│   ├── interim     <- Intermediate data that has been transformed. (added in .gitignore)
│   ├── processed   <- The final data sets for modeling. (added in .gitignore)
│   └── raw         <- The original, immutable data dump. (added in .gitignore)
│
├── docs                    <- Reports and presentations.
│   └── presentation.pdf    <- Final presentation for committee purposes.
|
├── notebooks                                       <- Jupyter notebooks.
│   ├── images                                      <- Data analysis graphs in .png format.              
│   ├── data_analysis.ipynb                         <- Jupyter notebooks containing the data analysis (Univariate Analysis, RFM Analysis and Exploratory Analysis).
│   ├── models.ipynb                                <- Jupyter notebooks containing the Machine Learning models (hyperparameter tuning, training, testing, model selection and model analysis).
│   └── preprocessing_feature_engineering.ipynb     <- Jupyter notebooks containing the data preprocessing and feature engineering.
|
├── .gitignore
|
├── INSTRUCTIONS.md
|
├── README.md
|
└── requirements.txt   <- The requirements file for reproducing the environment.

```
